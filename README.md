## ✨ Features

- 💱 Real-time currency conversion
- 🌍 Multiple currency support
- 🚩 Country flags for each currency
- 🔄 Swap currencies instantly
- ⚡ Auto update on input change (debounce)
- 🎯 Clean and modern UI
- 📱 Fully responsive design
